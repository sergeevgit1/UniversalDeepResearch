# Вклад во frontend Universal Deep Research

Код предназначен только для исследовательского прототипа.

## Как участвовать
1. Форкните репозиторий и клонируйте его:
   ```bash
   git clone <your-fork-url>
   cd apps/frontend
   ```
2. Установите зависимости:
   ```bash
   npm install
   # или
   yarn install
   ```
3. Вносите изменения по существующим паттернам, добавляйте тесты и обновляйте документацию.
4. Проверьте:
   - Компоненты в dev‑режиме
   - Отображение на разных разрешениях
   - Соблюдение требований доступности
5. Подайте PR с понятным названием и описанием изменений.

## Стандарты кода
- **TypeScript** в строгом режиме
- **React**: функциональные компоненты и хуки
- **Next.js**: конвенции App Router
- **CSS**: модули для стилизации
- Документация через JSDoc/комментарии
- Явная обработка ошибок и состояния ошибок
- Настройки через переменные окружения
- Соблюдайте WCAG и семантический HTML

## Полезные команды
```bash
npm run dev        # запуск dev-сервера
npm run build      # сборка production
npm run lint       # статический анализ
```
