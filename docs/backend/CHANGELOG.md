# История изменений

Ключевые обновления backend Universal Deep Research (UDR-B).

## [1.0.0] — 15.07.2025
### Добавлено
- Первая версия UDR-B
- REST API на FastAPI со стримингом ответов
- Интеллектуальные исследования с LLM и веб‑поиском
- Поддержка нескольких моделей (OpenAI, NVIDIA, локальный vLLM)
- Интеграция Tavily для поиска
- Управление сессиями и постоянными данными
- Фрейм рассуждений FrameV4
- Режим dry‑run с mock‑данными
- Гибкая конфигурация через переменные окружения
- Онлайн‑прогресс через Server‑Sent Events
- CORS для интеграции с frontend
- Логирование и трассировка
- Mock‑данные для разработки и тестов

### Технические возможности
- Эндпоинт `/api/research` для полного цикла исследования
- Эндпоинт `/api/research2` на базе FrameV4
- Модульная архитектура с конфигурируемыми компонентами
- Поддержка нескольких LLM‑провайдеров и моделей
- Конфигурация через окружение
- Скрипт установки для быстрого запуска
- Обработка ошибок и валидация

### Документация
